# 📸 Screenshots Folder

Place your dashboard screenshots and demo video here.

## Required Files

```
screenshots/
├── dashboard-main.png       (Main dashboard overview)
├── weather-panel.png        (Weather integration panel)
├── historical-trends.png    (Charts and analytics)
├── ml-performance.png       (Model training results)
└── demo-video.gif          (Optional: Animated demo)
```

## Quick Capture Guide

1. **Dashboard**: Open `index.html` → Capture main view
2. **Weather**: Scroll to weather section → Capture
3. **Trends**: Scroll to charts → Capture
4. **ML Results**: Open `AQI_ML_Training.ipynb` → Run Cell 5 → Capture plots

## Tools

- **Windows**: Press `Windows + Shift + S`
- **Mac**: Press `Cmd + Shift + 4`
- **Video**: Press `Windows + G` (Game Bar)

## Tips

- Use 1920x1080 resolution
- Save as PNG format
- Keep file sizes under 2MB
- Show real data (not placeholders)

Once you add the files, they'll automatically appear in the main README.md!
